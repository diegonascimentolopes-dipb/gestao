💰 Gestor Financeiro OfflineUm sistema de gestão financeira moderno, rápido e focado em privacidade total. Este gestor foi desenvolvido para rodar inteiramente no navegador do usuário, eliminando a necessidade de servidores externos ou bancos de dados na nuvem.🚀 Sobre o ProjetoO Gestor Financeiro Offline é uma ferramenta independente que utiliza o poder do LocalStorage para persistência de dados. Diferente de outros sistemas, suas informações financeiras nunca saem do seu dispositivo. É a solução ideal para quem deseja controle total sem abrir mão da segurança.🛡️ Privacidade em Primeiro LugarZero Backend: Não há bancos de dados externos. Seus dados moram no seu navegador.Totalmente Independente: Funciona sem conexão com a internet após o carregamento inicial.Código Aberto: Transparência total sobre como seus dados são manipulados.✨ Funcionalidades Principais📊 Relatórios Mensais: Filtros por mês e ano com resumo visual de gastos totais por integrante.✂️ Divisão por Porcentagem: Lógica avançada para dividir despesas entre pessoas de forma igualitária ou com porcentagens personalizadas.📅 Gestão de Parcelamento: Cadastro automático de compras parceladas para meses futuros.♻️ Backup e Restauração: Sistema de exportação e importação via arquivos .json para sincronização manual entre dispositivos.📱 Suporte PWA: Interface otimizada para mobile, permitindo a instalação como "Web App" na tela de início do seu celular.💼 Múltiplas Carteiras: Possibilidade de nomear o banco de dados local para separar finanças Pessoais de Profissionais.🛠️ Stack TecnológicaO projeto foi construído utilizando as tecnologias web mais estáveis e performáticas:HTML5: Estrutura semântica e suporte a Web Storage.CSS3 (Bootstrap 5.3): Interface moderna, responsiva e com estética de banco digital.JavaScript (ES6+): Lógica de processamento de dados e cálculos financeiros pura, sem frameworks pesados.Bootstrap Icons: Conjunto de ícones leves e elegantes.⚙️ Como Hospedar (GitHub Pages)Como o projeto é um arquivo único estático, a hospedagem no GitHub Pages é imediata e gratuita:Crie um novo repositório no seu GitHub chamado Gestor-Financeiro-Offline.Faça o upload do arquivo index.html (ou o nome do seu arquivo principal) para a branch main.No seu repositório, vá em Settings (Configurações).No menu lateral esquerdo, clique em Pages.Em Build and deployment > Branch, selecione a branch main (ou root) e clique em Save.Aguarde alguns segundos e o GitHub fornecerá um link (ex: https://seu-usuario.github.io/Gestor-Financeiro-Offline/).📖 Como UsarConfigurações: Vá na aba "Ajustes" para cadastrar os nomes dos membros, categorias e status.Lançar: Utilize a aba principal para inserir seus gastos diários ou parcelados.Relatórios: Filtre o período desejado para ver o resumo de quanto cada pessoa deve pagar.Extrato: Visualize o quadro individual de cada membro para conferência.Backup: Lembre-se de exportar seu JSON regularmente para garantir que não perderá os dados caso limpe o cache do navegador.📄 LicençaEste projeto está sob a licença MIT.
